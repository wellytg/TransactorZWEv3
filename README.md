# TRANSACTOR ZWE v3

A modern, lightweight financial calculator designed to help users in Zimbabwe navigate the complex landscape of Intermediated Money Transfer (IMT) Tax and varying bank charges.

Transactor ZWE v3 provides real-time calculations for:

- **POS/Swipe Charges:** Tiered calculations based on transaction amounts.
- **Zipit/Bank Transfers:** Estimates bank-specific transfer fees and minimum balance requirements.
- **Transaction Limits:** Built-in validation for Zipit ($20,000) and RTGS ($500,000) limits.
